# Calc

## Teste para vaga Pekus

### Overview
- `Calc/Components/Pages/Calculator.razor` carrega a página principal
    - Campos de valores, botões de operações, e lista de cálculos feitos
- `Calc/Features/CalculadoraAPI.cs` contém métodos para se conectar com a API em `http://pekus.ddns.net/calcapi/swagger/index.html`
- `Calc/Features/CalculadoraModel.cs` contém o modelo de um cálculo, e métodos utilitários

### Tecnologias usadas
- .NET SDK v9
- Framework Blazor
- IDE VSCodium
